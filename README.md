# Run the app locally

This contains everything you need to run the site locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. (Optional) Set the `GEMINI_API_KEY` in `.env.local`
3. Run the app:
   `npm run dev`
